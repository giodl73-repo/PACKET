# Connectivity Peer Panel (illustrative)

Archetype peer reviewers for PACKET research outputs — distinct from the parliament
(adversarial design review) and the editorial gate (form review). Each is an
archetype, not a named person.

| ID | Reviewer archetype | Expertise |
|---|---|---|
| R-P1 | Broadband policy researcher | Universal service, coverage mapping, BEAD-style programs |
| R-P2 | Network scientist | Internet topology, graph algorithms, resilience |
| R-P3 | Telecom economist | Benefit-cost, competition, demand/take-rate modeling |
| R-P4 | Network operations researcher | Performance measurement, peering, congestion |
| R-P5 | Digital-equity scholar | Adoption, affordability, the divide |

Assignment guide:

- Topology / resilience method → R-P2, R-P4
- Coverage / policy / benefit-cost → R-P1, R-P3
- Equity / adoption → R-P5, R-P1
- Tier/SLA + multi-scale methodology → R-P1, R-P2, R-P4
